# AI Mood Detector

## Introduction

AI Mood Detector is a web-based application that uses Artificial Intelligence to detect and understand human emotions through facial expressions. It can recognize emotions such as happy, sad, angry, surprise, disgust, fear, and neutral in real time.

## Problem

Some children find it difficult to understand human emotions, and parents may also struggle to explain these emotions clearly. This can affect communication, learning, and emotional development.

## Solution

This project provides an AI-powered system that detects facial expressions and displays the emotion using text and sound. This helps children learn and recognize emotions more easily in an interactive and engaging way.

## Features

* Real-time emotion detection using camera
* Image upload for mood detection
* Displays emotion in text and sound
* Supports multiple emotions (happy, sad, angry, etc.)
* Simple and user-friendly interface

## Technologies Used

* HTML
* CSS
* JavaScript
* Teachable Machine 

## How to Use

1. Open the website
2. Allow camera access OR upload an image
3. Click the detect button
4. View and listen to the predicted emotion

## Project Structure

* `index.html` – Main webpage
* `style.css` – Design and layout
* `script.js` – AI logic and functionality
* `model/` – Trained AI model files
